# Add-ons Overview

Recommended Blender add-ons for visual investigation work.

## Categories

### Photogrammetry & Scanning
Add-ons for 3D reconstruction from photos and scans.

### Measurement & Analysis
Tools for precise measurements and spatial analysis.

### Import/Export
Enhanced import and export capabilities for various file formats.

### Workflow Enhancement
Add-ons that improve efficiency and workflow.

### Visualization
Tools for better rendering and presentation.

## Installation Guide

### How to Install Add-ons

1. Download the add-on (usually a .zip file)
2. Open Blender → Edit → Preferences
3. Go to Add-ons tab
4. Click "Install..."
5. Navigate to the downloaded file
6. Select and click "Install Add-on"
7. Enable the add-on by checking the box

### Managing Add-ons

- Enable/disable in Preferences → Add-ons
- Update when new versions are available
- Read documentation for each add-on
