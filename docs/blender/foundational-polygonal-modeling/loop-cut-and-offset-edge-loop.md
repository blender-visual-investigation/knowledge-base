# Loop Cut & Offset Edge Loop

## Loop Cut
The Loop Cut tool allows you to split a loop of faces by inserting a new edge loop intersecting the chosen edge.

**Shortcut:** `Ctrl + R`

## Offset Edge Loop
This tool takes an existing edge loop and adds new edge loops on either side of it.
