# On Blender

Introduction to Blender and why it's valuable for visual investigation.

## What is Blender?

Blender is a free, open-source 3D creation suite that supports:

- 3D modeling
- Animation
- Rendering
- Compositing
- Motion tracking
- Video editing

## Why Blender for Visual Investigation?

### Open Source & Free

- No licensing costs
- Transparent algorithms
- Reproducible results
- Community-driven development

### Comprehensive Toolset

- **Photogrammetry**: Reconstruct 3D scenes from photographs
- **Camera Tracking**: Match virtual cameras to real footage
- **Precise Measurements**: Accurate scale and dimensional tools
- **Documentation**: Built-in rendering for reports and presentations

### Forensic Applications

- Crime scene reconstruction
- Accident analysis
- Architectural forensics
- Evidence visualization
- Timeline animations
- Spatial analysis

### Industry Standard Features

- Professional rendering engines
- Python scripting for automation
- Extensive add-on ecosystem
- Regular updates and improvements
