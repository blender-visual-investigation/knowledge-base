## Introduction

This knowledge base is designed to help you learn Blender for visual investigation and 3D reconstruction work. Whether you're just getting started or have some experience, you'll find practical guidance written with accessibility in mind. We aim to explain concepts clearly without oversimplifying the work, mixing everyday language with the technical terms you'll encounter in the field.

## Who Can Benefit

This knowledge base serves a wide range of people working with visual evidence and 3D tools:

- **Investigators and analysts** - Reconstructing scenes, visualizing evidence, and documenting spatial information
- **Journalists and researchers** - Using open-source methods (OSINT) to verify and analyze visual content
- **3D artists transitioning to investigation work** - Applying your modeling skills to real-world cases
- **Students and academics** - Learning methods for spatial analysis and visual documentation
- **Career changers and hobbyists** - Exploring how 3D tools can support investigative work

### Our Approach to Language

We've written this content to be as accessible as possible while respecting the professional nature of the work. You'll find:

- **Plain explanations first** - Core concepts explained in everyday language
- **Technical terms introduced gradually** - When specialized vocabulary is necessary, we define it clearly
- **Real-world context** - Examples and use cases that show why techniques matter
- **Academic rigor where it counts** - Methodology and documentation guidance for those who need defensible, reproducible work

The goal is simple: anyone motivated to learn can start here, and as you progress, you'll naturally pick up the professional terminology used in the field.
