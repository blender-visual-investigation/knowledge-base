# The 3D Cursor

The 3D Cursor is a powerful tool unique to Blender. It acts as a temporary anchor point in 3D space.

## Uses
- Spawning new objects at a specific location.
- Acting as a pivot point for transformations.
- Snapping objects to a specific point.

**Shortcut:** `Shift + Right Click` to place (by default).
