# Precise Snapping

Snapping allows you to align objects or components perfectly with others.

## Snapping Options
- **Increment**: Snaps to grid units.
- **Vertex**: Snaps to vertices.
- **Edge**: Snaps to edges.
- **Face**: Snaps to faces.

**Toggle Snapping:** `Shift + Tab`
