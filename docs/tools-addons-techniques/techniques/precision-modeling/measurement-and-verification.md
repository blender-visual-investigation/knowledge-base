# Measurement & Verification

Verifying the accuracy of your model is critical in forensic reconstruction.

## Measure Tool
The interactive Measure tool allows you to draw lines to measure distances and angles in the viewport.

## Edge Length Overlay
Displaying edge lengths in Edit Mode for real-time feedback.

## MeasureIt Add-on
Using the built-in MeasureIt add-on for rendering measurements and creating technical documentation.
