## How to Install Addons

**Drag & Drop**
Drag the `.zip` or `.py` file directly into the Blender window.

**Extensions Platform**
**Edit > Preferences > Get Extensions**. Search for the addon and click **Install**.

**Install from Disk**
**Edit > Preferences > Get Extensions**. Click the top-right arrow, select **Install from Disk**, and choose the file.
