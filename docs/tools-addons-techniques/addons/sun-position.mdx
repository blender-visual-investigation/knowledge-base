## Introduction

The Sun Position addon allows you to simulate real-world lighting conditions. You can set the time of day, date, and geographic location (latitude/longitude) to position the sun accurately. This is critical for forensic reconstruction where lighting conditions must match a specific time and place.

## Installation & Usage

### Installation
This addon is included with Blender.

1. Go to **Edit > Preferences > Add-ons**.
2. Search for **Sun Position**.
3. Enable the checkbox.

### Usage
Go to the **World Properties** tab in the Properties Editor. Scroll down to find the **Sun Position** panel. You can assign a Sun object and a Sky Texture, then adjust the time and location settings.
