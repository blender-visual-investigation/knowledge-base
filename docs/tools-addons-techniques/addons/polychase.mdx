## Introduction

*Note: Specific details for "PolyChase" were not found in standard repositories. This might be a custom or less common addon.*

Generally, "Chase" tools in Blender allow a camera to smoothly follow a target object with delay and damping. This is useful for creating third-person views, smooth fly-throughs, or following a moving subject without rigid mechanical movement.

## Installation & Usage

### Installation
1. Install the script/addon via **Edit > Preferences > Add-ons**.

### Usage
Select the camera, then the target object, and activate the tool from the addon's panel or menu.

## Summary

Camera chase tools add a layer of professionalism to animations by removing the robotic feel of standard parenting. They are essential for dynamic presentations.
