## Introduction

This addon allows you to import project files generated by the standalone **fSpy** application. fSpy is an open-source tool used to calculate camera parameters (focal length, orientation, position) from a single image. This is crucial for camera matching and compositing 3D objects into 2D photos.

## Installation & Usage

### Installation
1. Download the addon zip from the [official GitHub repository](https://github.com/stuffmatic/fSpy-Blender).
2. Go to **Edit > Preferences > Add-ons**.
3. Click **Install** and select the zip file.
4. Enable the addon.

### Usage
Go to **File > Import > fSpy (.fspy)** and select your project file. A new camera will be created with the correct settings.

---

## Summary

The fSpy importer is the bridge between the fSpy application and Blender. It is the standard workflow for single-image camera matching, making it easy to reconstruct scenes from photographs.
