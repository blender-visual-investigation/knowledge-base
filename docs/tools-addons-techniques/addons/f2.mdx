## Introduction

The **F2 add-on** is a free, built-in tool designed to significantly accelerate the **poly-modeling** workflow in Blender. It functions as a powerful extension of the default `F` key (fill) operation, enabling rapid, repetitive face creation with minimal, consistent keystrokes.

## Installation

The F2 add-on is included with recent Blender versions and only requires activation.

**Step-by-step installation:**

1.  Navigate to the **Edit** menu and select **Preferences**.
2.  In the Preferences window, select the **Add-ons** tab.
3.  Use the search field to find **"F2"**.
4.  Enable the add-on by checking the box next to **"Mesh: F2"**.

---

## Usage

Use the F2 add-on primarily in Edit Mode for filling gaps in mesh geometry.

### 1. Edge-Based Repetitive Filling

This method is efficient for filling open edge loops.

1.  Enter **Edit Mode** (Tab) on your mesh.
2.  Switch to **Edge Select** mode (press `2`).
3.  Select a **single edge** adjacent to the area you want to fill.
4.  Repeatedly press the **`F`** key to sequentially fill faces along the adjacent boundary.


### 2. Single Vertex Filling

This is useful for quickly building geometry from a starting point.

1.  Enter **Edit Mode** (Tab).
2.  Switch to **Vertex Select** mode (press `1`).
3.  Select a **single vertex** adjacent to the open area.
4.  Repeatedly press the **`F`** key. Each press will create a new face, generating a new vertex that attempts to align with the surrounding geometry.

### 3. Controlling Fill Direction

Use the mouse position to resolve ambiguity when filling.

1.  In **Edit Mode**, select the Edge or Vertex that is the input for the fill.
2.  Before pressing `F`, **hover the mouse cursor** over the side (left or right) where the new face should be created.
3.  Press **`F`** to generate the face in the direction indicated by the cursor.

## Summary

The F2 add-on significantly improves the face-filling process by substituting complex, disconnected key sequences with fast, repetitive presses of the **`F`** key. This focus on repetitive keystrokes streamlines retopology and poly-modeling workflows, yielding substantial time savings once integrated into the user's modeling mindset.
