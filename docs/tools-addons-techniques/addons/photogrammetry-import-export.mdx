## Introduction

This refers to a class of addons (like the popular **Photogrammetry Importer**) that allow you to bring data from photogrammetry software into Blender. This includes camera positions, sparse point clouds, and dense meshes. It is essential for integrating 3D scans into your workflow.

## Installation & Usage

### Installation
1. Download the appropriate addon (e.g., [Photogrammetry Importer](https://github.com/SBCV/Blender-Addon-Photogrammetry-Importer)).
2. Install via **Edit > Preferences > Add-ons**.

### Usage
Use **File > Import** and select the format corresponding to your photogrammetry software (e.g., `.sfm`, `.xml`, `.abc`).

---

## Summary

Photogrammetry importers are vital for reconstruction workflows. They allow you to verify the accuracy of your scans and use the solved camera data for animation or compositing.
