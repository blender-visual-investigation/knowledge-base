## Introduction

MeasureIt is an addon designed to display measures in the 3D Viewport. It can show segment lengths, angles, and areas. The measurements are rendered as an overlay and can be rendered to an image, making it perfect for technical drawings or architectural visualization.

## Installation & Usage

### Installation
This addon is included with Blender.

1. Go to **Edit > Preferences > Add-ons**.
2. Search for **MeasureIt**.
3. Enable the checkbox.

### Usage
Find the **MeasureIt Tools** in the Sidebar (N-panel) under the **View** tab. Select elements in Edit Mode and click the "Segment" or "Angle" buttons to add a measurement.
